# Music Festival Project V1

A dynamic landing page for a music festival, built with modern web development practices for efficient styling, automation, and interactivity.

## Tech Stack
- NPM
- SASS
- Gulp
- JavaScript

## Key Features

### NPM
- Dependency management
- SASS compilation
- Watch mode

### Gulp
- Setup and configuration
- SASS compilation
- Watch mode

### SASS
- Variables and files
- `@forward`, `@use`, `@import`
- `&` operator
- Responsive mixins

### JavaScript
- DOM manipulation
- Event handling

## Notes

To rebuild the Node modules: 
```
npm install
```
To run Gulp:
```
npm run dev
